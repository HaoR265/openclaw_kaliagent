# Kaliclaw 历史文档归档

这些文档仍有价值，但默认只作为：

1. 历史背景
2. 设计演进记录
3. 旧方案对照

不再作为当前 Kaliclaw 的正式权威入口。

## 使用规则

1. 新功能说明优先写到正式入口文档。
2. 历史文档只做引用修复、事实校正和归档说明，不再继续扩写成正式规范。
3. 如果历史材料需要继续演进，优先在正式入口重写，而不是直接复活旧标题。
